# Security Policy

## Supported versions

Only the latest published version of `allmd` receives security fixes. Please
upgrade to the newest release before reporting an issue.

## Reporting a vulnerability

Report vulnerabilities privately through GitHub's security advisories:

1. Go to the [Security tab](https://github.com/mblode/allmd/security/advisories/new).
2. Click **Report a vulnerability** and describe the issue, including steps to
   reproduce and the affected version.

Please do not open a public issue for security reports.

## Response

You can expect an initial acknowledgement within 7 days. Once a fix is ready it
is released as a new version and the advisory is published with credit to the
reporter unless you ask to remain anonymous.
