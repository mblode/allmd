---
title: PowerPoint
description: Convert PowerPoint presentations to markdown.
---

## Usage

```bash
allmd pptx <file> -o output.md
```

## Supported Formats

- `.pptx`

## Example

```bash
# Convert a presentation to markdown
allmd pptx slides.pptx -o slides.md

# Output to stdout
allmd pptx deck.pptx
```
