---
title: CSV
description: Convert CSV and TSV files to markdown tables.
---

## Usage

```bash
allmd csv <file> -o output.md
```

## Supported Formats

- `.csv`
- `.tsv`

## Example

```bash
# Convert a CSV to a markdown table
allmd csv data.csv -o table.md

# Convert a TSV file
allmd csv spreadsheet.tsv -o spreadsheet.md
```
